# Study With Me
A React and Websockets app that allows users to coordinate study sessions and chat during breaks.

### UP & RUNNING
* `npm install`
* `npm start`
* visit `http://localhost:8080/`

### DEPENDENCIES
* babel-core
* babel-loader
* babel-preset-es2015
* babel-preset-react
* css-loader
* express
* left-pad
* moment
* node-sass
* react
* react-dom
* sass-loader
* socket.io
* style-loader
* uuid
* webpack
